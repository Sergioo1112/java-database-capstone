# User Stories

## Patient User Story

**Title:**
_As a Patient, I want to book and manage my appointments, so that I can easily schedule visits with doctors._

**Acceptance Criteria:**
1. Patient can view available appointments.
2. Patient can book an appointment.
3. Patient can cancel or reschedule an appointment.

**Priority:** High  
**Story Points:** 3  
**Notes:**
- Patients can only manage their own appointments.

## Doctor User Story

**Title:**
_As a Doctor, I want to manage my availability and view my appointments, so that I can organize my work schedule._

**Acceptance Criteria:**
1. Doctor can set available time slots.
2. Doctor can view upcoming appointments.
3. Doctor can update appointment status after a consultation.

**Priority:** High  
**Story Points:** 5  
**Notes:**
- Doctors can only access their own schedules and appointments.

## Admin User Story 1
