# User Stories

## Admin User Stories

- As an Admin, I want to create, update, and delete user accounts so that I can manage system access.
- As an Admin, I want to assign roles (Admin, Doctor, Patient) so that users have the correct permissions.
- As an Admin, I want to view all appointments so that I can monitor clinic activity.

---

## Patient User Stories

- As a Patient, I want to register an account so that I can access the clinic portal.
- As a Patient, I want to book an appointment with a doctor so that I can receive medical care.
- As a Patient, I want to cancel or reschedule my appointments so that I can manage my schedule.
- As a Patient, I want to view my appointment history so that I can keep track of my visits.

---

## Doctor User Stories

- As a Doctor, I want to view my daily appointments so that I can organize my work.
- As a Doctor, I want to update my availability so that patients can book available time slots.
- As a Doctor, I want to view patient information before appointments so that I can provide better care.
- As a Doctor, I want to create prescriptions for patients so that treatment information is recorded.
